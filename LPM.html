Milestone 2

Machine Learning Logic

Azzi and I had decided to split the job by software and hardware, to ensure that we are on track and able to meet the deadline.

For machine learning, I had to research and find the most suitable regression for the project. Thankfully I was able to find a cheatsheet that microsoft had produce and it contains lot of useful information that is being simplified for us to understand.

For abnormal detection we used one-class support vector machine as it aims to determine a suitable region in the input data space, Ï‡, which includes most of the samples drawn from an unknown probability distribution, P. It detects objects that resemble training samples.
For Threshold readings we used poisson regression as it as it is a generalized linear model form of regression analysis used to model count data and contingency tables. Poisson regression assumes the response variable Y has a Poisson distribution, and assumes the logarithm of its expected value can be modeled by a linear combination of unknown parameters.
